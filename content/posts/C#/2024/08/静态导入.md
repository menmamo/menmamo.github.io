## 静态导入

用法：在引用命名空间时，在using关键字后面加入static关键词

作用：无需指定类型名称即可访问其静态成员和嵌套类型

好处：节约代码量，可以写出更简洁的代码

```c#
using static UnityEngine.Mathf;
public class Test3
{
    public void Test()
    {
       Max(10, 20); 
    }
}
```
