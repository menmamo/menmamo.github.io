### 分部类

- 把一个类分成几部分来声明
- 使用关键字partial
- 分部描述一个类
- 增加程序的扩展性
- 分部类可以写在多个脚本文件中
- 分部类中部可以有重复的成员
- 分部类的访问修饰符要一致

```c#
partial class Student{
	public bool sex;
	public string name;
}
partial class Student{
	public bool sex;
	public int id;
}
```
