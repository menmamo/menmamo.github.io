### 异常捕获

我们通过异常捕获 可以避免当代码报错时造成程序卡死

```C#
//基本语法
try
{
   //填写希望进行异常捕获的代码 如果报错 会执行catch中的代码来捕获异常
}
catch
{
    //可通过catch(Exception e)可通过e得到具体的错误信息
}
//可选部分
finally
{
    //最后执行的代码 不管有没有出错都会执行
}
```
