### 内部类

- 在类的内部 声明的类
- 使用时要用 包裹者点出自己

```c#
class Person{
    public int age;
    public string name;
    //内部类
	public class Body{
        Arm leftArm;
        Arm rightArm;
        class Arm{
            
        }
	}
}
//使用
Person.Body body = new Person.Body();
```
