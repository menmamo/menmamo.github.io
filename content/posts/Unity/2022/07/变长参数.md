参数前面加关键字params，参数必须为数组，传入的参数会添加进数组中。

变长参数必须作为最后一个参数使用，且只能有一个变长参数。

```c#
public int Sum(params int[] arr){
    int sum = 0;
    for(int i = 0; i<arr.Length; i++){
        sum += i;
}
    return sum;
}
Sum(1,2,3,4,5,6);
```
